# Шаг 11 — Финальный прогон

**Предыдущий:** [step-10-certificates.md](step-10-certificates.md)

## Зачем

Проверить **весь** Educa LMS end-to-end: каталог, CMS, студент, прогресс, викторины, сертификаты.

## Подготовка

- Сервер: `python manage.py runserver`
- Пользователи: `admin`, `teacher`, `student`
- Тестовый курс **Python Basics** с модулями, контентом и викториной

## ✅ Типичные ошибки (если что-то сломано)

| Симптом | Причина | Решение |
|---------|---------|---------|
| `NoReverseMatch` в шапке | URL не подключён | Шаги 4, 7 |
| 403 на `/course/mine/` | Нет группы Instructors | Шаг 2 |
| CSS не грузится | Кэш браузера | Ctrl+F5; `courses/static/css/base.css` |
| Файлы не открываются | Нет MEDIA | Шаг 1 |
| Quiz 404 | Нет моделей / миграции | Шаг 3 |
| Progress всегда 0% | Нет services / models | Шаги 5–6 |
| Сертификат не выдаётся | Не все модули пройдены | Шаг 8, 10 |
